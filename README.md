# **EGuBa(이거봐)**

## 🔎 프로젝트 개요
#### 친구들과 함께 하는 영상 공유 플랫폼
> ***기획 배경***<br>
> 기존의 회의 플랫폼과는 다르게 친구들과 함께 영화, 드라마 등 영상을 함께보고 채팅도 하면서 여가 시간을 보낼 수 있는 가상 공간을 제공하고자 플랫폼을 만들었습니다.<br/>
